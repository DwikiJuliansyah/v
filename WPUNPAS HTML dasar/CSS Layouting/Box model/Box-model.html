Box Model

  • setiap elemen di halaman berada di dalam sebuah box(kotak)
  • kita bisa mengatur ukuran dan posisi kotak tersebut
  • kita bisa memberi warna/gambar sebagai background kotak tersebut

CSS box model mendifenisikan 'kotak' yang dihasilkan oleh sebuah elemen, lalu menampilkannya sesuai dengan format visualnya.

CSS box model terdiri dari margin, border, padding, dan content.

CSS box model

margin

area transparan di sekitar kotak(di luar border)

padding

area transparan di dalam kotak(di dalam border)

border

batas di sekiling konten dan padding

content

konten sebenarnya ada di dalam box, bisa berupa teks atau gambar.
