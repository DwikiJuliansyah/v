Anatomi

Syntax CSS

selector{ property: value; }

Contoh :

h1{ color: blue; }

Selector

  • Digunakan untuk memilih dan memanipulasi elemen spesifik pada HTML

  • Elemen HTML dipilih berdasarkan tag, id, class bahkan pola atau pattern.

  • Semakin kompleks struktur HTML, selektor juga bisa semakin kompleks/spesifik
